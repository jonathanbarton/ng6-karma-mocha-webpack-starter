# Overview

## 1. Setup

### Tools / Frameworks Used
 - webpack: Bundling
 - npm: package / dependency management, run-scripts
 - angular 1.4 + ui-router
 - karma + mocha + chai + istanbul
 - protractor
 - fs-extra: generator scripts

### Installation

```
npm install

```

## 2. Commands


```
# Build
npm run build

# Unit Tests
npm run test

# e2e Tests
protractor protractor.conf.js

# Serve & Watch
npm run serve

```
